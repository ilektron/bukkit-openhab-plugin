## Setup
- Download plugin from [here](https://github.com/ilektron/bukkit-openhab-plugin/releases/download/1.10/OHMinecraft.jar). 
- Add jar to Server->plugins.

Set listening port in Server->plugins->OHMinecraft->config.yml
